# カーボンクレジットからクーリングクレジットへの移行シミュレーション

**概念的・反実仮想・因果シミュレーション — 2015年〜2035年**

> *カーボンクレジットからクーリングクレジットが実現した場合、何が変わっていたか？*

---

## このシミュレーションについて

本モジュールは、**概念的・因果的・インデックスベースの反実仮想モデル**です。

これは精密な気候予測ではありません。
℃、W/m²、GtC/yr、USDなどの物理単位で結果を出力するものではありません。
すべてのインデックスは0〜100に正規化された例示的な代理指標です。

目的は、一つの構造的な問いを考察することです：

> **カーボンクレジットへ向かった同規模、またはそれ以上の資金が、クーリングクレジット（測定可能な物理的冷却・自然冷却機能回復・水循環回復・土壌回復・森林蒸散・海洋循環支援・熱負荷低減）へ向けられていたなら、主要な気候システム指標の軌跡はどうなっていたか？**

---

## 七つの核心命題

このシミュレーションは七つの構造的主張に基づいています：

1. **カーボンクレジットとクーリングクレジットは異なるものを評価する。** カーボンクレジットはCO₂削減・相殺単位を測定する。クーリングクレジットは測定可能な物理的冷却、熱負荷低減、自然冷却機能回復を測定する。

2. **カーボンクレジットは炭素会計である。** 主な効果は排出量帳簿へのものである。物理的な熱負荷、水循環、土壌水分、森林蒸散、海洋熱への直接効果は弱い。

3. **クーリングクレジットは熱会計であり自然冷却機能会計である。** 主な効果は直接的な物理的冷却、水循環回復、土壌水分、森林蒸散、都市冷却、海洋・流域循環支援である。

4. **同規模の資金がカーボンオフセットから測定可能な冷却へ移動すれば、期待される影響は異なる。** すべての次元で大きいとは限らないが、構造的に異なる——炭素会計ではなく熱負荷・自然冷却機能・水循環を標的とする。

5. **クーリングクレジットは排出削減の代替ではない。** 脱炭素化は引き続き必要である。クーリングクレジットはカーボンクレジット資金が届かない不足していた層を追加する。

6. **クーリングクレジットは不足していた層を追加する：測定可能な熱負荷低減と自然冷却機能回復。** 地球温暖化は熱として体験される——炭素帳簿の数字としてではなく。不足していた層は熱を直接対処するメカニズムである。

7. **このモデルは概念的・反実仮想的であり、精密な気候予測ではない。** すべてのインデックス値は、文書化された世界的なトレンドと方向的に整合させた例示的な代理指標である。

## シミュレーション概要

### 期間

2015〜2035年

### シナリオ

| ID | 名称 | 説明 |
|---|---|---|
| **A** | カーボンクレジット基準 | カーボンクレジット中心の金融。物理的冷却・自然冷却機能回復は弱いまま。 |
| **B** | 等規模クーリングクレジット移行 | Aと同規模の総資金を、測定可能な物理的冷却へ転換。 |
| **C** | 大規模クーリングクレジット移行 | クーリングクレジットがESG・保険・適応・防災資金を追加で引き寄せる。 |
| **D** | ハイブリッド経路 | 排出量会計にカーボンクレジットを維持し、第二層としてクーリングクレジットを追加。 |

---

### 熱負荷低減経路

![熱負荷低減経路](outputs/heat_load_reduction_pathways.png)

- **シナリオA（赤）**：炭素会計が物理的な熱を直接低減しないため、熱負荷が急上昇。
- **シナリオC（青）**：大規模な物理的冷却投資により、熱負荷の軌跡が大幅に緩和。
- A〜Cのギャップが、炭素会計と冷却機能投資の構造的な差異を表す。

---

### 自然冷却機能回復インデックス

![自然冷却機能回復インデックス](outputs/natural_cooling_recovery_index.png)

地球の自然冷却機能——森林蒸散・土壌水分・水循環・海洋循環・生物学的冷却——はシナリオAで低下する。クーリングクレジット資金が物理的冷却回復へ向かうシナリオB・C・Dでは回復する。

---

### 水循環・陸域冷却回復

![水循環回復インデックス](outputs/water_cycle_recovery_index.png)

陸域冷却回復の三次元：
- **水循環回復**：流域回復、降水パターン回復。
- **土壌水分回復**：土壌保水能——地表面の主要な冷却バッファー。
- **森林蒸散回復**：潜熱移送を通じた最も重要な陸域冷却メカニズム。

クーリングクレジット資金がこれらのシステムへ流れるシナリオB・C・Dでは三次元すべてが大きく回復する。

---

### 海洋熱とエルニーニョリスク緩和

![海洋熱とエルニーニョリスク](outputs/ocean_heat_and_el_nino_risk_index.png)

**重要な注意事項**：クーリングクレジットはエルニーニョを止めることはできない。エルニーニョは太平洋の海面水温ダイナミクスによって駆動される大気・海洋結合気候モードである。

このシミュレーションがモデル化しているのは、以下を通じた**エルニーニョ関連の熱リスク増幅の潜在的緩和**である：

- 地球システムの背景熱負荷の低減
- 土地・水の冷却バッファー（土壌水分、森林蒸散）の改善
- 地表面冷却改善を通じた部分的な海洋熱緩和
- 都市冷却による局地的な熱ストレス増幅の低減

エルニーニョが低い背景熱負荷の上に重なった場合、高い背景熱負荷の上に重なった場合より、その熱リスク増幅は小さくなると考えられる。これが`el_nino_heat_risk_moderation_index`が表すものである。

![クーリングクレジット規模効果](outputs/cooling_credit_scaling_effect.png)

## カーボンクレジット対クーリングクレジット：構造比較

| 次元 | カーボンクレジット | クーリングクレジット |
|---|---|---|
| **資金標的** | 炭素会計とオフセット | 物理的冷却と自然冷却機能回復 |
| **主要メカニズム** | CO₂排出量バランス | 熱負荷低減と冷却機能回復 |
| **熱への直接効果** | 弱い——排出削減を通じて間接的 | 強い——物理的冷却を直接標的 |
| **水循環効果** | 弱い | 強い——流域・土壌・蒸散に資金投入 |
| **土壌水分効果** | 弱い | 強い——土壌有機物・水分回復に資金投入 |
| **森林効果** | 部分的（炭素のための植林） | 全体的（蒸散冷却を優先） |
| **海洋効果** | 非常に弱い | 中程度——背景熱低減を通じて |
| **エルニーニョ効果** | 非常に弱い | 部分的——熱リスク増幅の緩和 |
| **都市冷却** | 弱い | 強い——都市熱低減がコア標的 |
| **不足していた層** | 物理的冷却と自然冷却機能回復 | （これが不足していた層そのもの） |
